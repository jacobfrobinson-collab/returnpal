Configure Returns Settings

Set your Amazon return address to send unfulfillable inventory to ReturnPal

Use these steps in Amazon Seller Central so your FBA unfulfillable returns are sent to our facility. This helps clients change their return address to ReturnPal.

  1. Log in to Amazon Seller Central
  2. Go to Settings → Fulfillment by Amazon
  3. Scroll to Automated Unfulfillable Settings and click Edit
  4. Ensure Enable is toggled on for Automated Unfulfillable Settings
  5. Ensure Refurbishment is selected for Enroll in value recovery options
  6. Scroll to the Removals section and make sure Return is toggled on
  7. Set Schedule to Weekly (on the 8th, 15th, 22nd and 28th)
  8. Enter our address as the return address (use your personal or company name for the return address):
