Marketplace / eBay — all clients in one file

Upload a .csv or .xlsx where each row is one sale. The Client ID column must match either the client’s dashboard ID (e.g. 14 or 0014) or their Old Client ID from the table below (e.g. AC, SM). Matching rows are saved as sales on that client’s dashboard immediately (visible after refresh).

Rows with a missing Client ID or an ID that does not match anyone yet can be saved to Pending import rows below (when the option is on), so you can apply them after the client exists or after you set their Old Client ID.

Columns: Client ID (or Old Client ID / legacy_client_id), sold_date (YYYY-MM-DD), item_name, quantity, earnings. Optional: order_number.
